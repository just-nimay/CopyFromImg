Это приложение разработано для операционной системы Windows
версия python 9.7.3

INSTALL

1. Установите Tessetact: https://github.com/UB-Mannheim/tesseract/wiki
2. Создайте виртуальное окружение, и уже в нем устанавливайте зависимости с помощью команды: pip install -r requirements.txt

USAGE

запустите программу командой: python main.py

после нажатия "ctrl + shift + s" вам нужно будет выделить область, в которой будет находиться текст, который вам нужно скопировать. После, текст покажется в окне терминала, и скопируется в буфер обмена.

что бы поменять язык, который будет распознаваться, (по умолчанию Русский) зажмите "ctrl + shuft + x": напишите в консоль доступный язык (rus - Русский, eng - Английский).

что бы выйти из программы, нажмите esc.
